Contact and Support
===================

If you need help with Globus Compute, there are several ways to get support or ask questions.

GitHub Issues
-------------

For bug reports, feature requests, or general questions, please `open an issue in our GitHub repository <https://github.com/globus/globus-compute>`_.

Email Support
-------------

For direct support, you can email our team at support@globus.org or `create a support ticket <https://support.globus.org/hc/en-us/requests/new>`_.

Globus Discuss Mailing List
---------------------------

For broader discussions, announcements, and community support, you can join and post to the `Globus Discuss mailing list <https://groups.google.com/a/globus.org/g/discuss>`_.
